# gqlgen-lambda
제목은 [gqlgen으로 GraphQL만들기]로 하겠습니다. 근데 이제 AWS Lambda를 곁들인.

### 사용한(할) 기술
 - Gqlgen
 - AWS(Lambda, Api-gateway, AppSync ...)
 - Entgo Orm
 - MySQL On Docker(using Makefile)
